# 性癖バトル ローグライクデッキ構築

Slay the Spire 的なデッキ構築ローグライク。
ボイロキャラクターの性癖で戦い、相手の理性を0にしたら勝ち。

---

## 基本用語

| 用語 | 元の概念 | 説明 |
|---|---|---|
| 理性 | HP | 0になったら負け |
| 冷静 | ブロック | 被ダメを軽減。ターン終了時にリセット |
| 熱量 | 筋力 | 全ダメージに加算される永続バフ |
| 動揺 | 脆弱+弱体 | 被ダメ+25%・与ダメ-25% |
| ダークサイド | 廃棄 | 使用後デッキから除外 |
| 異端 | - | 対応レリックなしではプレイ不可 |
| 品性 | エナジー | 毎ターン3回復 |
| 性癖 | レリック | パッシブ効果アイテム |

## カードの種類

### キャラクターカード
- コスト0、場は1枠のみ
- 新しいキャラを出すと前のキャラが退場（捨て札へ）
- **出す効果**（プレイ時1回）と**在場効果**（場にいる間毎ターン）を持つ
- 葵↔茜のみ例外：互いを出したとき退場キャラが手札に戻る（バウンス）
- かのん↔咲はカード交代ではなくネコ耳による**変身**（後述）

### アクションカード
- 品性を消費してプレイ
- 表面・裏面があり、強化すると裏面になる（名前も変わる）

### 異端カード
- デッキに入るがプレイ不可（グレーアウト）
- 手動で捨て札には送れる
- 対応レリックを持つと解放される
- 通常カードの1.5〜2倍の効果

## ゲームフロー

- 全20フロア、10階・20階にボス
- 各フロアクリア後、ランダム3択でノードを選ぶ

| ノード | 報酬 |
|---|---|
| 通常戦闘 | カード3択 |
| エリート | レリック3択 |
| ショップ | カード削除・異端カード指名買い・理性回復 |
| 回復 | 理性回復 |
| 強化 | カード1枚を強化（表→裏） |

### コイン
- フロアクリアで1コイン獲得
- ショップでのみ使用

| 値段 | 商品 |
|---|---|
| 2コイン | 理性回復 |
| 3コイン | カード削除 |
| 4〜5コイン | 異端カード指名買い |

## ボス

### 1ボス（10階）：再生型
**鉄壁の理性** 理性100

毎ターン理性+12回復。攻撃は毎ターン10。
→ 1ターンに12以上出し続けないと削り負ける。瞬間火力が問われる。

有利：速攻コンボ、姉妹ループ、ダークサイド反転  
不利：沼堕ち、耐えて反転

### 2ボス（20階）：鈍感力剥がし型
**業の体現** 理性150

初期冷静50（毎ターン-10）。攻撃毎ターン12。
→ 6ターン目まで冷静が張られていてほぼ削れない。
→ 溜め切ってから爆発させる。

有利：沼堕ち、耐えて反転、冷静爆発  
不利：速攻コンボ
